Formularios Dinámicos y Fetch

El objetivo de este proyecto es desarrollar un sitio web dinámico que integre:

  • Formularios interactivos para agregar y mostrar productos.
  • Hacer uso de una API externa para generar contenido dinámico (en este caso fakestoreapi).
  • Maquetación con Grid aplicando diseño responsivo para móvil, tablet y PC.
